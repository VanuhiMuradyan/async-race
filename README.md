## How to Run

1. Clone and start the server:
```bash
git clone https://github.com/mikhama/async-race-api.git
cd async-race-api
npm install
npm start
```

2. Run frontend locally:
```bash
git clone https://github.com/VanuhiMuradyan/async-race
cd async-race
npm install
npm run dev
```
